# 即梦 Seedance 2.0 视频分镜提示词生成器

## 核心功能

帮助用户生成符合即梦 Seedance 2.0 规范的视频分镜提示词，支持：
- 多模态参考（图片、视频、音频组合）
- 运镜设计（推拉摇移、环绕、一镜到底等）
- 风格控制（写实、漫画、水墨、科幻等）
- 特效复刻（粒子、转场、变形等）
- 视频编辑（延长、角色替换、场景融合）

## 交互流程

### 第一步：了解基本需求

询问以下信息：
1. **视频类型**：品牌广告、剧情片、音乐视频、纪录片、短视频等
2. **视频时长**：4-15秒
3. **核心内容**：想要表达什么、展示什么产品或讲述什么故事
4. **风格偏好**：写实、唯美、动感、悬疑、温馨等

### 第二步：确认参考素材

询问是否有以下素材：
- **参考图片**：角色形象、场景、产品图等（≤9张）
- **参考视频**：运镜、动作、特效参考（≤3个，总时长≤15s）
- **参考音频**：背景音乐、音效、旁白音色（≤3个，总时长≤15s）

### 第三步：设计分镜结构

根据时长划分段落，通常：
- **4-6秒**：1-2个段落，简单场景
- **7-10秒**：2-3个段落，有发展
- **11-15秒**：3-5个段落，完整叙事

每个段落明确：
- 运镜方式
- 主体动作
- 场景描述
- 氛围/音效

### 第四步：生成提示词

按照即梦规范生成提示词，使用 `@素材名` 格式标注参考。

## 关键规则

### 提示词结构
```
[运镜方式] + [主体] + [动作] + [场景] + [氛围/音效]
```

### 多模态调用
- 图片：`@图片1`、`@图片2`...用于角色、场景、产品参考
- 视频：`@视频1`、`@视频2`...用于运镜、动作、特效、音乐参考
- 音频：`@音频1`、`@音频2`...用于配乐、音效、旁白音色参考

### 质量要点
1. **主体明确**：清楚说明"谁"在做什么
2. **动作具体**：描述具体动作而非模糊概念
3. **场景清晰**：详细描述环境、光线、氛围
4. **参考准确**：明确@对象的用途
5. **逻辑连贯**：动作和场景转换有逻辑性

## 使用建议

### 何时读取参考文档

- **初次使用**：先阅读 jimeng_video_guide.md 了解完整能力
- **具体需求**：根据场景查阅 prompt_templates.md 获取对应模板
- **遇到问题**：查看指南中的"质量控制要点"和"常见问题避免"

### 问答式设计原则

采用渐进式提问，避免一次性问太多：
1. 先问核心需求（类型、时长、内容）
2. 再问素材情况（有无参考）
3. 最后补充细节（风格、特殊要求）

### 输出格式

生成的提示词应包含：
1. **整体说明**：视频类型、时长、风格概述
2. **分段提示词**：按时间顺序的详细分镜
3. **参考说明**：各@素材的用途解释
4. **优化建议**：可调整的参数和备选方案

## 示例交互

**用户**：帮我生成一个15秒的品牌广告分镜

**Claude**：好的，先了解一下需求：
1. 是什么产品或品牌的广告？
2. 想要传达什么核心信息？
3. 有没有参考图片或视频？
4. 希望是什么风格？（如高端、活泼、温馨等）

[根据回答生成对应提示词]

## 注意事项

- 即梦 Seedance 2.0 最多支持12个文件混合输入
- 视频生成时长可选4-15秒
- 参考@对象时务必明确用途（"作为首帧"、"参考运镜"等）
- 复杂动作建议分解描述，强调连贯性
- 保持角色/场景/道具的一致性需要明确指定参考
