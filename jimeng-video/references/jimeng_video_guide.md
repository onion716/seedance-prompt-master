# 即梦 Seedance 2.0 视频创作完整指南

## 一、核心能力概览

### 1.1 多模态输入限制
- **图片**：≤ 9 张（支持首帧、尾帧、主体参考、场景参考）
- **视频**：≤ 3 个，总时长不超过 15s
- **音频**：≤ 3 个，总时长不超过 15s（MP3格式）
- **文本**：自然语言描述
- **生成时长**：4-15s 可选
- **文件总数**：≤ 12 个（混合输入上限）

### 1.2 交互方式
通过 `@素材名` 来指定每个图片、视频、音频的用途：
- `@图片1` 作为首帧
- `@视频1` 参考镜头语言
- `@音频1` 用于配乐

## 二、视频分镜核心要素

### 2.1 镜头语言（运镜）

#### 基础运镜
| 运镜类型 | 描述 | 应用场景 |
|---------|------|---------|
| 推镜头 | 镜头向主体推进 | 突出情绪、特写、强调重点 |
| 拉镜头 | 镜头远离主体 | 展示环境、结束感 |
| 摇镜头 | 镜头左右/上下转动 | 展示场景、跟随移动 |
| 移镜头 | 镜头平行移动 | 跟随主体、横移展示 |
| 跟镜头 | 跟随主体移动 | 追逐、运动场景 |
| 升降镜头 | 镜头垂直升降 | 展示空间关系 |

#### 高级运镜
| 运镜类型 | 描述 | 示例 |
|---------|------|-----|
| 环绕镜头 | 围绕主体旋转拍摄 | "环绕镜头展示角色360度" |
| 鱼眼镜头 | 超广角变形效果 | "鱼眼镜头向下窥视" |
| 希区柯克变焦 | 推拉同时进行 | "惊恐时希区柯克变焦" |
| 一镜到底 | 连续不剪辑的长镜头 | "一镜到底，从街头到屋顶" |
| 主观视角 | 角色第一人称视角 | "第一人称视角推近" |
| 机械臂运镜 | 精准多角度跟随 | "机械臂多角度跟随视线" |
| 急速运镜 | 快速运动切换 | "急速的左右环绕运镜" |

### 2.2 景别设置

| 景别 | 画面范围 | 应用 |
|-----|---------|-----|
| 远景/大远景 | 展示环境、地貌 | 开场、场景建立 |
| 全景 | 人物全身 + 环境 | 展示动作、位置关系 |
| 中景 | 人物半身 | 对话、常规叙事 |
| 近景 | 人物胸部以上 | 情绪表达、对话 |
| 特写 | 细节局部 | 强调重点、情绪高潮 |
| 大特写 | 极致细节 | 特定物体、微观表现 |

### 2.3 角度选择

| 角度 | 效果 | 应用 |
|-----|------|-----|
| 平视 | 自然、客观 | 常规叙事 |
| 俯视 | 弱化、渺小感 | 展示环境、弱势 |
| 仰视 | 强大、压迫感 | 英雄感、威胁 |
| 荷兰角 | 倾斜、不安 | 紧张、混乱 |
| 过肩镜头 | 对话关系 | 双人对话 |

## 四、动作与情绪表达

### 4.1 动作描述要点
- **明确主体**：谁在做动作
- **动作顺序**：动作的先后次序
- **动作细节**：具体怎么做
- **动作连贯**：动作之间的衔接

### 4.2 常见动作类型

| 动作类型 | 示例提示词 |
|---------|-----------|
| 日常动作 | "优雅地晒衣服"、"用力抖衣服" |
| 运动动作 | "奔跑"、"跳跃"、"翻滚" |
| 打斗动作 | "打斗场面"、"挥剑"、"格挡" |
| 表情动作 | "心虚的表情左右看"、"满足的表情" |
| 互动动作 | "拥抱"、"握手"、"递过物品" |

### 4.3 情绪表达

| 情绪 | 表现方式 | 示例 |
|-----|---------|-----|
| 惊恐 | 希区柯克变焦、慌乱动作 | "惊恐时希区柯克变焦" |
| 愤怒 | 面部特写、激烈动作 | "咬牙切齿"、"愤怒说话" |
| 温馨 | 柔光、慢动作 | "温馨的室内"、"欢快跑过来" |
| 紧张 | 快速剪辑、急促运镜 | "急速逃亡"、"惊慌撞倒" |
| 喜悦 | 明亮色调、轻快节奏 | "欢快的音乐"、"笑容洋溢" |

## 五、音频与节奏

### 5.1 音频使用方式
- **背景音乐**：设定氛围和节奏
- **音效**：增强真实感（脚步声、环境音）
- **旁白/对话**：推动剧情发展
- **音乐卡点**：画面与音乐节奏同步

### 5.2 节奏控制

| 节奏类型 | 特征 | 应用 |
|---------|------|-----|
| 快节奏 | 快速切换、短镜头 | 动作戏、高潮 |
| 慢节奏 | 长镜头、慢动作 | 情绪戏、氛围 |
| 卡点节奏 | 与音乐同步 | 音乐视频、广告 |
| 渐变节奏 | 从慢到快/快到慢 | 过渡、铺垫 |

## 六、高级创作技巧

### 6.1 视频延长

**延长类型**：
- **向前延长**：在原视频前增加内容
- **向后延长**：在原视频后增加内容
- **中间插入**：在原视频中间插入新场景

**提示词示例**：
- "将@视频1延长5s"
- "向前延长10s，内容为..."
- "在@视频1和@视频2之间加一个场景"

### 6.2 角色替换与编辑

**替换场景**：
- 人物替换："将@视频1中的女生换成戏曲花旦"
- 局部修改："将@视频1女人发型变成红色长发"
- 完整复刻："动作完全模仿原视频"

### 6.3 一致性控制

**保持一致的方法**：
- **人物一致**：指定"参考@图1的人物形象"
- **场景一致**：明确指定场景参考
- **道具一致**：说明道具在镜头中的保持
- **风格一致**：统一指定风格参考

### 6.4 多参考组合

**组合方式**：
- 多图场景拼接："上方@图2，左边@图3，右边@图4"
- 多视频融合："参考@视频1的运镜+@视频2的动作"
- 图视频结合："@图1的人物形象+@视频1的运镜方式"

## 七、提示词结构模板

### 7.1 基础模板
```
[运镜方式]，[主体动作]，[场景描述]，[氛围/风格]，[特殊要求]
```

### 7.2 分段模板
```
0-X秒画面：[运镜] + [主体] + [动作] + [场景]
X-Y秒画面：[运镜] + [主体] + [动作] + [场景]
...
```

### 7.3 参考模板
```
使用@图片X的[元素]，参考@视频X的[运镜/动作/特效]，[具体描述]
```

### 7.4 完整示例模板
```
[运镜]，[主体]参考@图片X，[动作]参考@视频X，
场景为[详细描述]，背景音效/音乐参考@音频X，
整体风格为[风格描述]，时长为X秒。
```

## 八、常见应用场景提示词示例

### 8.1 品牌广告
```

```

### 8.2 剧情片

参考@视频1的运镜和节奏，

## 九、质量控制要点

### 9.1 生成高质量视频的关键

1. **明确的主体**：清楚说明"谁"在做什么
2. **具体的动作**：描述具体动作，而非模糊概念
3. **清晰的场景**：详细描述环境、光线、氛围
4. **准确的参考**：明确@对象的用途
5. **连贯的逻辑**：动作和场景转换要有逻辑性

### 9.2 常见问题避免

| 问题 | 避免 | 正确做法 |
|-----|------|---------|
| 主体模糊 | "一个人在走" | "@图1的男人在走廊里走" |
| 动作不清 | "做一些动作" | "先跳跃后翻滚，保持连贯" |
| 场景缺失 | "在某个地方" | "在19世纪的伦敦大街上" |
| 参考混乱 | "用这个和那个" | "@图1作为首帧，参考@视频1的运镜" |
| 时序混乱 | "然后、然后、然后" | "0-3秒...，4-8秒...，9-13秒..." |

## 十、创作流程建议

### 10.1 问答式创作流程

1. **了解需求**
   - 视频类型（广告/剧情/音乐视频等）
   - 时长要求
   - 风格偏好
   - 核心信息传达

2. **确认素材**
   - 是否有参考图片/视频/音频
   - 素材数量和用途

3. **设计分镜**
   - 根据时长划分段落
   - 确定每段的运镜、动作、场景
   - 确保连贯性和节奏感

4. **撰写提示词**
   - 按模板组织语言
   - 明确标注@参考对象
   - 检查逻辑和细节

5. **优化调整**
   - 根据生成结果迭代
   - 调整细节和参数
