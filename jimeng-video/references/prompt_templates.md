# 即梦视频提示词模板库

## 按视频类型分类

### 1. 品牌广告类

#### 产品展示
```
特写展示@图片1的[产品名称]，
镜头环绕展示全貌，突出产品细节和质感，
背景为[场景描述]，光线[光线描述：如"温暖柔和"或"高级质感"]，
背景音恢宏大气，
时长X秒。
```

#### 故事化广告
```
@图片1的[角色]在[场景]中，
镜头[运镜方式：如"小幅度拉远并跟随"]，
[剧情描述：角色遇到问题/发现产品/使用后变化]，
最后产品特写，广告语"[广告词]"以字幕形式出现，
整体风格[风格描述]。
```

### 2. 剧情片类

#### 开场场景
```
以@图片1为首帧画面，
镜头[运镜方式]展示[场景环境]，
@图片2的[角色]在[做什么]，
光线[描述]，氛围[氛围描述]，
时长X秒。
```

#### 冲突场景
```
参考@视频1的运镜和节奏，
@图1的角色和@图2的角色在[场景]中相遇，
镜头[具体运镜：如"侧面跟拍"或"急速环绕"]，
[角色动作和对话]，
情绪[情绪描述：如"紧张"/"愤怒"]。
```

#### 情绪场景
```
@图1的角色[动作/状态]，
镜头推近至面部特写，
[表情和情绪变化描述]，
背景音[声音描述]，
整体风格[风格描述：如"唯美"/"压抑"]。
```

### 3. 音乐视频（MV）类

#### 卡点切换
```
参考@视频1的音乐节奏进行卡点，
@图片1-@图片N根据音乐节奏变换景别和角度，
每次切换伴随[转场效果：如"重影闪烁"/"无缝渐变"]，
画面更[动感/梦幻/有张力]，
可根据音乐改变景别和光影变化。
```

#### 一镜到底MV
```
一镜到底，镜头[运动轨迹]，
从[起始场景]开始，
[途经场景1]→[途经场景2]→...→[结束场景]，
@图片1的角色全程在画面中，
参考@视频1的音乐和节奏。
```

### 4. 纪录片类

#### 人文纪录片
```
电影级写实风格，2.35:1宽银幕，24fps，
细腻的画面风格，
拍摄"[主题]",
展示[具体内容：环境/人物/活动]，
旁白音色参考@视频1，
结合旁白阐述[主题意义]。
```

#### 产品纪录片
```
以@图片1的[产品]为拍摄对象，
镜头[运镜方式]展示产品细节和制作过程，
背景为[工作场景]，
真实环境音+旁白，
整体质感[描述：如"专业"/"匠心"]。
```

### 5. 短视频/社交媒体类

#### 换装视频
```
@图片1的角色在镜子前，
参考@图片2-@图片N的服装样式，
每次换装伴随[转场效果：如"切镜"/"闪烁"]，
镜头参考@视频1的运镜和节奏，
整体风格[风格描述]。
```

#### 手势舞/舞蹈
```
@图片1的角色参考@视频1的舞蹈动作，
镜头[运镜方式：如有节奏的推拉摇移]，
在[场景]中活力十足地表演，
音乐参考@视频1。
```

## 按运镜方式分类

### 推镜头
```
镜头从[起始景别]推近至[结束景别]，
展示[主体]的[细节/表情]，
[场景/氛围描述]。
```

### 拉镜头
```
镜头从[起始景别]拉远至[结束景别]，
从[主体]扩展展示到[整体环境]，
[结束画面描述]。
```

### 摇镜头
```
固定机位，镜头[左右/上下/对角]摇动，
从[起始画面]摇到[结束画面]，
展示[场景关系/空间变化]。
```

### 跟随镜头
```
镜头跟随@图片1的[角色]在[场景]中移动，
[运镜细节：如"保持中景"、"侧面跟拍"]，
[角色动作描述]。
```

### 环绕镜头
```
镜头[360度/半圆]环绕@图片1的[角色/主体]，
展示[主体]的[多角度细节/全貌]，
[场景/氛围描述]。
```

### 一镜到底
```
一镜到底的[运动轨迹]镜头，
从[起始点]→[途经点1]→[途经点2]→...→[终点]，
[途中发生的动作/事件]，
保持镜头连贯不剪辑。
```

## 按特殊技巧分类

### 视频延长
```
将@视频1[向前/向后]延长X秒，
[延长部分的具体内容描述]，
保持与原视频的[风格/节奏/角色]一致。
```

### 角色替换
```
将@视频1中的[原角色]换成@图片1的[新角色]，
[保持/修改]原视频的[动作/运镜/场景]，
[其他要求]。
```

### 多场景拼接
```
镜头[运动轨迹]，
展示多个场景：
- [场景1参考@图1]：[内容]
- [场景2参考@图2]：[内容]
- [场景3参考@图3]：[内容]
场景间[转场方式]自然过渡。
```

### 风格迁移
```
@图片1的主体，
参考@视频1的[运镜/特效/节奏]，
风格为[目标风格描述]，
[其他细节要求]。
```

## 分段式提示词模板

### 三段式（15秒）
```
0-5秒画面：[运镜] + [主体] + [动作] + [场景]，[氛围/音效]
6-10秒画面：[运镜] + [主体] + [动作] + [场景]，[氛围/音效]
11-15秒画面：[运镜] + [主体] + [动作] + [场景]，[结束画面/字幕]
```

### 五段式（15秒）
```
0-3秒画面：[开场描述]
4-6秒画面：[发展描述]
7-9秒画面：[高潮/转折描述]
10-12秒画面：[延续/结果描述]
13-15秒画面：[结尾/字幕]
```

## 快速组合模板

### 基础结构
```
[运镜方式] + [主体] + [动作] + [场景] + [氛围/音效] + [特殊要求]
```

### 带参考结构
```
使用@图片X的[元素]，参考@视频X的[运镜/动作/特效/音乐]，
[具体描述内容]，[风格/氛围]。
```

### 完整结构
```
[运镜] + @图片X的[主体] + [动作描述] + [场景参考@图X]
+ [氛围/光线] + [音效/音乐参考@音频X] + [时长] + [特殊要求]
```
